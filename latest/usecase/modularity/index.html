Measuring the modularity

In this example, we will use label_propagation to optimize the modularity of a network. The data are from the mcmullen dataset.

julia> using EcologicalNetwork

julia> # Get the data in an object
       N = mcmullen();

julia> # We will start with a random assignment of species within modules
       L = rand(1:richness(N), richness(N));

julia> # We can now start a repeated number of attempts to find the best partition. If
       # julia is started in parallel, this will use all assigned CPUs.
       M = modularity(N, L, replicates=100);
