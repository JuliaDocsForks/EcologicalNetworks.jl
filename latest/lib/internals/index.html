# EcologicalNetwork.motif_internal! — Method.

Internal motif calculations

The two arguments are N the network and m the motif adjacency matrix (as a DeterministicNetwork). The two matrices must have the same size. The function returns a vectorized probability of each interaction being in the right state for the motif, i.e. P if m is 1, and 1 - P if m is 0.

# EcologicalNetwork.pdi — Method.

Paired Differences Index for specificity. This function will range the values of each row, so that the strongest link has a value of one.

julia> pdi(vec([0.0 0.0 1.0]))
1.0

julia> pdi(vec([1.0 1.0 1.0]))
